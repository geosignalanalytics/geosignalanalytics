DAS Data Analysis & Interpretation

Program Introduction

This program focuses on the analysis and interpretation of Distributed Acoustic Sensing (DAS) data for subsurface monitoring and seismic applications.

Program Scope

Participants will explore DAS-specific data characteristics, processing challenges, and advanced analysis techniques used in research and field applications.

Program Topics

  • DAS acquisition principles
  • DAS signal characteristics and noise sources
  • Processing and denoising of DAS data
  • Seismic and ambient noise applications
  • Case studies from real DAS deployments

Target Audience

Researchers, engineers, and practitioners working with DAS or large-scale sensing systems.
