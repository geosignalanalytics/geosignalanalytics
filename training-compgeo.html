Computational Geophysics

\ \

\ This program provides a solid foundation in computational methods for geophysical modeling,\ simulation, and interpretation, with a strong emphasis on seismic applications and subsurface imaging.\

\ \

Scope and Objectives

\

\ The training focuses on numerical methods and computational workflows used in modern geophysics,\ including forward modeling, inverse problems, and large-scale data analysis. Emphasis is placed on\ linking physical theory with efficient computational implementation.\

\ \

Key Topics

\
    \
  • Numerical methods for geophysical modeling
    • \
  • Forward and inverse problems
    • \
  • Computational seismic imaging
    • \
  • Optimization and regularization techniques
    • \
  • High-performance and large-scale computing concepts
    • \
\ \

Target Audience

\

\ Graduate students, early-career researchers, and professionals seeking strong computational\ foundations in geophysics.\
