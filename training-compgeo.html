DAS Data Denoising: Guided Unsupervised Deep Learning Network

Project Structure

  • asset: folder containing logo
  • data: folder containing data
  • notebooks: set of Jupyter notebooks reproducing the experiments in the paper
  • Matlab_CWT_Version: more stable version for the 2D CWT using Matlab
  • outputs: denoised data obtained by the proposed framework

Getting Started

To ensure reproducibility, use the DASDL.yml file to create the environment:

./install_env.sh

After installation, activate the environment:

conda activate DASDL

Scripts (Fully based on Python)

Go to the notebooks folder and run the file named DASDL_Main. The denoised outputs will appear in the outputs folder.

Scripts (CWT based on Matlab)

This is an alternative method using the 2D CWT in Matlab (more stable, less signal leakage):

  1. Run Prepare_CWT.m to obtain band-pass filter data and CWT scale.
  2. Run the Python file DASDL_Main.
  3. Outputs are saved in the outputs folder.

Disclaimer

All experiments were conducted on an Intel(R) Xeon(R) CPU @ 2.10GHz with a single NVIDIA GEForce RTX 3090 GPU. Different setups may require different configurations.
