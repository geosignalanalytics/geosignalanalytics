Signal Processing for Geophysical Applications

A core research and training program of the GeoSignal Research Institute

Program Introduction

This program covers fundamental and advanced signal processing techniques commonly used in geophysical data analysis, with applications to seismic and DAS data.

Program Scope

Participants will learn how to analyze, filter, and enhance geophysical signals while preserving physically meaningful information. Both classical and modern approaches are discussed.

Program Topics

  • Time-frequency analysis
  • Filtering and noise attenuation
  • Multichannel signal processing
  • Transform-based methods
  • Applications to seismic and DAS data

Target Audience

Students, researchers, and professionals working with geophysical or seismic datasets.
